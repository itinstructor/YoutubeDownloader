# Python YouTube Downloader
A Python project using pytube to download video or audio from YouTube.

This project uses the <a href="https://pytube.io/en/latest/">pytube </a> library.


## Youtube Audio Downloader Version History
- (02-15-23) V2 Add menu to continue downloading another or exit.
- (02-03-23) V1 Download an mp3 audio file from a YouTube video link.
